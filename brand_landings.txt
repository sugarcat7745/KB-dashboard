# 네이버 브랜드검색 등 '수동 점검' 랜딩 목록
# ─────────────────────────────────────────────
# 네이버 브랜드검색 소재의 랜딩(클릭 시 이동 주소)은 광고 API로 자동 추출이 안 된다
# (/ncc/ads 응답에 썸네일 이미지와 소재ID만 있고 랜딩 URL이 없음).
# 그래서 여기에 직접 적어두면 매시 자동 점검에 포함된다.
#
# 규칙:
#   - 한 줄에 URL 1개 (http/https 로 시작)
#   - 실제 클릭 시 이동하는 주소를 그대로 (utm 파라미터가 있으면 포함해서)
#   - '#' 뒤는 메모(주석)
#
# 예시:
#   https://lawfirmkb.com/?utm_source=naver&utm_medium=brand&utm_campaign=brand_search   # 브랜드검색 메인
#
