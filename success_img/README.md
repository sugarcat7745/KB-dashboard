# 성공사례 결과 도장 이미지

업체가 만든 결과별 도장 이미지를 이 폴더에 **`결과명.png`** 형식으로 넣으면,
성공사례 게시 시 자동으로 파일첨부됩니다(목록 카드 썸네일 = 이 이미지).

- 파일명 = 결과 문구 그대로. 예: `승소.png`, `불송치.png`, `기소유예.png`, `법정구속 면함.png`
- 동의어는 대표 표기 한 장만: 무혐의·불입건 → `혐의없음.png` / 약식 벌금형 → `약식명령.png` / 화해 성립 → `조정 성립.png`
- 없는 결과는 자동으로 '본문만 게시'(첨부 없음)로 처리됩니다.

권장 규격: 세로형(기존 성공사례와 동일, 예 600×769), png.
