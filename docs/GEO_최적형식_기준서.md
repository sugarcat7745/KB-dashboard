# 법률 Q&A GEO 최적 형식 기준서 (근거 종합)

> 목적: 생성형 AI(ChatGPT·Perplexity·AI Overviews·네이버 큐)가 **인용·노출하기 좋은** 법률 Q&A를 만들기 위한 근거 기반 기준.
> 출처: ① GEO 학술연구·엔진 인용 메커니즘 ② 한국 AI검색·YMYL·변호사 광고규정 ③ 실제 한국 로펌 글 15건+ 크롤·분석. (문서 끝 출처)
> 이 문서가 QnA 생성 프롬프트·레퍼런스·검수의 기준(single source)이다.

---

## 0. 한 줄 결론
**법률 Q&A GEO = "첫 줄 직답 + 조문·수치·판례 근거 + 표/체크리스트 + FAQ + 실명 변호사·검토일 + 최신성" 이고, 한국은 여기에 '변호사 광고규정 금지어 배제'와 '네이버 생태계 병행'이 필수다.**
세 리서치(학술·한국채널·실제로펌글)가 이 결론에 **독립적으로 수렴**했다 — 신뢰도 높음.

---

## 1. 근거로 검증된 GEO 기법 (학술 — 가장 단단함)

프린스턴 GEO 논문(KDD 2024, 유일한 동료심사 실험)에서 인용 가시성을 올린 상위 기법:
1. **인용문 추가**(+~41%) 2. **통계·수치 추가**(+~33%) 3. **출처 인용**(+~28%)
- ⭐ **"법률·정부(Law&Government)" 도메인에서는 '통계·수치 추가'가 최고 성능** — 법률 글엔 구체 수치·조문·기준을 박는 게 GEO에 제일 잘 먹힌다는 실측.
- ❌ **키워드 반복(스터핑)은 baseline보다 나쁨** — 옛 SEO식은 GEO에서 역효과.
- ⚠️ 논문 절대%는 통제환경 상대이득 → 방향(구체·인용·수치 우대)은 신뢰, 수치는 참고.

**법률(YMYL) 신뢰 신호(단일 최대 레버):** **실명·자격 변호사에 글 귀속 + 약력 링크 + 최종 검토일**. 1차 출처(법령·판례) 인용.

## 2. 한국 채널 특성 (반드시 반영)

**채널이 둘로 갈린다 — 신호가 다르다:**
| | 네이버(통합검색·큐) | 글로벌 AI(ChatGPT·Perplexity·AI Overviews) |
|---|---|---|
| 인용 소스 | **네이버 안 정보만** 요약·인용 | 웹 전체 + **제3자 디렉토리(로톡·대한변협) 인용 밀도** |
| 핵심 신호 | C-Rank(=출처 신뢰, 한 주제 꾸준히), 제목 앞 키워드, h2/h3, 1,500자+, 직답 먼저 | 직답 문단·자기완결 청크, 스키마, AI크롤러 허용(GPTBot/ClaudeBot), 최신성 |
| 우리 할 일 | 네이버 상위노출·마이비즈니스 완성 | 스키마 + 크롤 허용 + 로톡·변협 프로필 완성 |

**⚠️ 변호사 광고규정 — 콘텐츠에서 반드시 배제(위반 시 징계 리스크):**
- 우월성 단정: **최고·유일·1위·No.1**
- 결과 보장: **승소 보장·100% 무죄·전액 환불·승소율/석방율**
- 염가 강조: **무료 상담·업계 최저가**
- **"○○ 전문 법무법인/전담팀/전담센터"** — '전문'은 전문등록 개인 변호사만, **법인·팀은 불가**
- 전관 암시(부장판사/검사 출신 영향력 오인), 별점·등급 수치화
- **의무: 모든 게시글에 광고책임변호사 실명 표시.**

## 3. 실제 상위 로펌 글의 표준 골격 (15건 크롤 결과)

열어본 상위 로펌 글(대륜·화온·로엘·번화·YK·대세 등)이 **거의 동일한 골격**을 공유 — 사실상 한국 법률 SEO 표준 템플릿:

```
제목(질문형 또는 키워드형 + "2026 최신/총정리")
 └ 핵심요약 박스(결론 선행, 역피라미드) + 손실회피 훅 + 신뢰신호
   ├ 정의 (무엇인지)
   ├ 요건·기준 (법조문 번호 + 수치 [표])
   ├ 절차 (단계 리스트)
   ├ 피의자/피해자·상황별 분기
   ├ 대응 전략 ("혼자 대응 vs 변호사 선임" [비교표])
   ├ FAQ 3~5개 (실제 검색질의 Q + 2~4문장 직답 A)
   └ CTA (본문 흐름 안 끊고 말미·고정영역에)
분량: 단일쟁점 2,500~4,000자 / 총정리형 6,000자 내외
톤: 정보성 70~85% : 홍보성 15~30%
```

**상·하위를 가르는 결정적 변별점 = 근거 밀도:**
- 최상위(번화·대세): 조문 번호 + **실제 판례번호·대법원/casenote 링크**(번화 법령 11·판례 3).
- 중위: 조문은 인용하나 판례는 "대법원 판례에 따르면"으로 뭉뚱그림.
- 하위(안티패턴): 조문·판례 번호 없이 일반론만 — 읽기 좋아도 신뢰 약함.

**차별화 포인트(우리가 이길 수 있는 것):** ⭐ **"법정형(이론) vs 실무상 실제 선고범위(경험)" 분리 제시** — 독자가 가장 원하는 정보(화온의 강점). ⭐ **증거 확보 체크리스트**(대륜) 같은 실행가능 정보.

**안티패턴(하지 말 것):** 근거 없는 일반론 / 키워드 스터핑 제목 / **지역만 바꾼 양산 복제** / 홍보가 본문 흐름 끊기 / FAQ를 영업멘트로 오염 / 감정 스토리 과다 도입.

## 4. ⭐ 종합 — KB QnA에 적용할 최적 형식

세 리서치를 합친 **우리 표준**(현재 5섹션 골격을 유지·강화하는 방향):

1. **제목** = `키워드 변호사 | 구체 질문?` (템플릿 "○○의 처벌과 대응은?" 금지). 필요 시 "2026년 기준" 등 최신성 신호.
2. **핵심요약(직답)** = 결론 먼저 2~3줄, 조문/수치 포함. (=GEO BLUF, 논문·한국채널·로펌글 모두 1순위)
3. **핵심 사항** = 정의 + **정확한 조문 번호 + 구체 수치/기준**(법률 도메인 GEO 최고 기법). 가능하면 **표**(처벌기준·비율 등).
4. **필수 주의 사항** = 흔한 실수·함정 + ⭐ **법정형(이론) vs 실무 실제결과(경험) 분리**.
5. **실제 대응 순서** = 단계 리스트 + 필요 시 **증거/서류 체크리스트**.
6. **변호사 선임 이유·KB 강점** = **각 2~3문장으로 짧게**(홍보 15~30% 유지, 흐름 안 끊기).
7. **FAQ 3~5개**(신규 권장) = 실제 검색질의 Q + 2~4문장 직답. (구글 FAQ 스니펫·롱테일 대응. 소스: 우리 `consult_raw` 상담원문·문의시트 검색키워드에서 실제 질의 재활용 가능)
8. **판례 근거**(신규 권장, 상위품질 진입) = 조문뿐 아니라 **검증된 판례번호**. 단 **환각 방지 필수** — 현재 조문 web_search 검증처럼 판례도 검증된 것만.
9. **신뢰·규정** = 실명 변호사·검토일 표기(YMYL + 광고책임변호사 의무), **광고규정 금지어 자동 배제**.
10. **문단은 짧고 독립적으로**(청크), 단정 대신 "~할 수 있습니다" + 이유.
