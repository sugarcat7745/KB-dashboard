# QnA 생성 — GEO 리서치 & 카테고리 레퍼런스

이 문서는 KB QnA 원고 생성 규칙의 **근거**를 정리한 것이다.
다른 로펌 글 분석 + GEO(생성형 AI 최적화) 리서치 결과를 담고, **그 결론이 코드 어디에 반영됐는지**를 연결한다.

> 중요: 이 리서치는 AI가 런타임에 읽는 데이터가 아니라, **생성 프롬프트 규칙(app.py 상수)으로 압축돼** 반영돼 있다.
> 별도로 우리 우수 원고는 BigQuery `qna_reference`에 축적돼 생성 시 예시로 참고된다(RAG).

---

## 1. GEO 원리 (생성형 AI가 인용하기 좋은 글)

생성형 검색(ChatGPT·Gemini·Perplexity·구글 AI Overviews·네이버 큐)이 **인용·발췌하기 좋은** 글의 특징:

| 요인 | 설명 | 우리 반영 |
|---|---|---|
| **결론부터 직답** | 질문에 곧바로 답, 각 문단이 그 자체로 완결된 답 | 핵심요약 3줄 + 섹션별 직답 |
| **통계·수치** | 법률 도메인에서 특히 인용률↑(기간·비율·금액·요건 수치) | `[수치]` 지시 + 표 |
| **출처 인용** | 1차 공식 출처(법령·판례) 명시 | 검증 조문 + law.go.kr 링크 |
| **구조화(Q&A·목록·표)** | FAQ, 순서 목록, 비교표는 추출이 쉬움 | FAQ 마이크로데이터 · `<ol>` · 표 |
| **정확한 개체명·용어** | 모호한 일반론보다 정확한 법령명·기관명 | 충실성 규칙 |
| **E-E-A-T / YMYL** | 법률=민감분야 → 전문성·신뢰 신호가 인용에 유리 | (실명 저자·스키마는 업체 영역) |

**안티패턴(하지 말 것):** 키워드 반복, 과장·보장 표현('최고/유일/1위/승소보장/무료'),
결론 없는 장황한 서론, 홍보 위주(정보:홍보 = 대략 8:2가 이상적).

## 2. 다른 로펌 글 표준 골격 (크롤 분석)

국내 로펌 QnA/블로그의 잘 쓰인 글의 공통 뼈대:

```
제목(질문형) → 핵심 요약(직답) → 정의/요건 → (표: 요건·기준 대조)
→ 절차/대응 순서(목록) → 분기별 유의사항 → FAQ → 상담 안내(절제)
```

- **차별화 요소**: 판례번호·조문 인용(신뢰), 구체 수치(금액·기간), 실제 상황 예시.
- **정보:홍보 비율**: 정보 70~85% : 홍보 15~30%. 홍보는 절제할수록 신뢰↑.
- **KB 적용**: 위 골격을 5섹션(핵심 사항·필수 주의 사항·실제 대응 순서·변호사 선임 이유·KB 강점)으로 고정.

## 3. KB QnA 확정 형식 (게시글에 실제로 나가는 골격)

1. **표(한눈에 보기)** — 요건·기준·수치 비교(정보성 우선, 홍보성 비교표 지양)
2. **핵심 요약 답변** — 직답 3줄
3. **상세 5섹션** — 핵심 사항 / 필수 주의 사항 / 실제 대응 순서(순서형은 `<ol>`) / 변호사 선임 이유 / KB 강점
4. **FAQ** — 실검색형 Q&A 3~5개 (schema.org/FAQPage 마이크로데이터)
5. **관련 법령 (출처)** — 검증 조문 law.go.kr 링크
6. **관련 질문** — 같은 분류 내부 링크

## 4. 카테고리별 내용 방향 (13개)

`app.py`의 `QNA_PROFILES`에 코드화됨. 요지:

| 분류 | 독자·관점 | 처벌 프레임 | 핵심 주제 |
|---|---|---|---|
| 형사 | 피의자/피해자·방어대응 | 다룸 | 법정형·양형(합의·초범)·수사/재판 절차 |
| 성범죄 | 피의자/피해자 | 다룸 | 법정형·신상등록·취업제한·합의·무고 |
| 학교폭력 | 학생·학부모 | 조건부 | 학폭위 조치·생기부·불복 절차 |
| 음주운전·교통사고 | 운전자/피해자 | 조건부 | 수치별 처벌·면허구제·합의·손해배상 |
| 민사·행정 | 청구인/상대방 | **금지** | 청구권·손해액·입증·절차·비용 |
| 이혼·가사 | 당사자·가족 | **금지** | 청구권·분할비율·양육·청구기한·조정 |
| 소년범죄 | 소년·보호자 | 조건부 | 보호처분·형사구분·전과·절차 |
| 행정소송 | 처분받은 개인/사업자 | **금지** | 위법성·집행정지·제소기간·구제 |
| 금융범죄 | 피의자/피해자 | 다룸 | 법정형(특경법)·피해회수·성립요건 |
| 건설·부동산분쟁 | 도급/수급·임대차 | **금지** | 청구·항변·하자입증·시효·절차 |
| 소액및손해배상 | 개인 채권자 | **금지** | 소액소송·지급명령·집행·비용 |
| 회생·파산 | 채무자 | **금지** | 자격·요건·면책·압류중지·비용 |
| 외국인·출입국 | 외국인·관계자 | 조건부 | 체류자격·비자·강제퇴거·난민/귀화 |

**처벌 '금지' 분류**에서는 형사 개념(처벌·형량·구속·기소·전과·벌금)을 쓰지 않는다(`_qna_forbid`).

## 5. 키워드 충실성 (자주 나는 오류 방지)

키워드가 가리키는 **사건 구도(인물·성별·입장·행위)**를 모델이 임의로 바꾸지 못하게 한다(`QNA_FIDELITY`).
- 흔한 반전 쌍: 상간남/상간녀, 고소인/피고소인, 채권자/채무자, 임대인/임차인, 매도인/매수인, 원고/피고, 도급인/수급인, 가해/피해.
- 복합 키워드는 요소를 빼지 않는다. 모호하면 지어내 단정하지 않는다.

## 6. 코드 연결 지도

| 리서치 결론 | 반영 위치(app.py) |
|---|---|
| 카테고리별 내용 방향 | `QNA_PROFILES` |
| GEO 형식(직답·수치·출처) | `QNA_GEO_RULES`, `_qna_answer_prompt` |
| 키워드 충실성 | `QNA_FIDELITY` |
| 비형사 형사개념 금지 | `_qna_forbid` |
| 5섹션·FAQ·표·표현제약 | `_qna_answer_prompt` |
| FAQ 마이크로데이터 | `_qna_faq_html` |
| 순서형 → `<ol>` | `_qna_as_ordered` |
| 조문 출처 링크 | `_qna_lawlinks_html` |
| 관련 질문 내부링크 | `_qna_related_html` |
| 우수 원고 예시 참고(RAG) | `qna_reference`(BQ) · `qna_reference_for` · `_qna_ref_brief` |
| 지역/일반 반반 키워드 | `qna_reco_keywords` |

> 템플릿(업체) 영역 개선(JSON-LD·meta description·canonical·실명 저자)은 `docs/GEO_홈페이지업체_요청서.md` 참고.
